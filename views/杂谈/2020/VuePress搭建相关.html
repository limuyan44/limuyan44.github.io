# VuePress搭建博客

博客经历了wordpress,hexo,最终决定换成了vuepress,一方面为了顺便熟悉一下vue,另一方面重要的是我希望博客最重要的聚集在内容上，因此对于md的要求希望能够自定义程度高一些，vuepress能够在md里面使用vue的一些东西，对于写作来说自由度很高。

# 安装

前提条件

VuePress 需要 Node.js (opens new window) >= 8.6

  1. 创建并进入一个新目录
mkdir vuepress-starter && cd vuepress-starter
1
  1. 使用你喜欢的包管理器进行初始化
yarn init # npm init
1

  1. 将 VuePress 安装为本地依赖

    我们已经不再推荐全局安装 VuePress

yarn add -D vuepress # npm install -D vuepress
1

注意

如果你的现有项目依赖了 webpack 3.x，我们推荐使用 Yarn (opens new window) 而不是 npm 来安装 VuePress。因为在这种情形下，npm 会生成错误的依赖树。

  1. 创建你的第一篇文档
mkdir docs && echo '# Hello VuePress' > docs/README.md
1

  1. 在 package.json 中添加一些 scripts (opens new window)

    这一步骤是可选的，但我们推荐你完成它。在下文中，我们会默认这些 scripts 已经被添加。

{
    "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
    }
}
1
2
3
4
5
6
  1. 在本地启动服务器
yarn docs:dev # npm run docs:dev
1

注意

vuepress不限制你的文章在哪个文件夹里面，只要在docs里面即可。
